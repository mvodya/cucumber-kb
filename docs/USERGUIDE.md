# Руководство пользователя CucumberKB UI

Необходимо подготовить данные для спецификации - [модель](/docs/SPECIFICATION.md#модель) и [базу знаний](/docs/SPECIFICATION.md#база-знаний) и [запрос](/docs/SPECIFICATION.md#запрос).

Нажать кнопку "Отправить в решатель".

В поле Решение заполнится данными.
